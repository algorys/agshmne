Téléchargements


Télécharger l'archive qui vous concerne et décompresser la dans le répertoire de votre choix :

  • Archive au format .zip
  • Archive au format .tar.gz

Pour démarrer le jeu, lancer le script correspondant à votre OS :

  • Pour Windows, exécutez agshmne.bat
  • Pour Linux, exécutez les commandes suivantes :
    • chmod +x agshmne.sh
    ./agshmne.sh
